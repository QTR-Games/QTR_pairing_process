# Screenshot Baseline Matrix

1. 1920x1080 at 100% scaling
2. 2560x1440 at 125% scaling

1. Main workspace idle
2. Main workspace with generated pairings
3. Main workspace with each active sort mode

## Naming Convention

Use:

<surface>__<resolution>__<scale>__<state>.png

Examples:

- main_workspace__1920x1080__100__idle.png
- data_management__2560x1440__125__advanced_expanded.png
- rating_dialog__1920x1080__100__default.png

## Capture Notes

- Capture after all widgets finish layout.
- Ensure no clipped labels/buttons.
- Use same sample dataset between before/after runs.
- Record timestamp and commit hash with each batch.
